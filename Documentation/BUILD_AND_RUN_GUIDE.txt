PREREQUISITES
------------------------------------------------------------
- Android Studio Meerkat (2024.3.1) or newer
- JDK 17 (bundled with Android Studio)
- Android SDK 35 (API 35)
- Internet connection (for first-time Gradle sync only)
- Physical Android device or Android Emulator (API 26+)


STEPS TO BUILD AND RUN
------------------------------------------------------------

1. EXTRACT THE ZIP
   - Extract "SmartStock_MCO1_SourceCode.zip" to any folder.
   - You should see a "SmartStock" folder containing the project files.

2. OPEN IN ANDROID STUDIO
   - Launch Android Studio.
   - Select File > Open (or "Open" from the Welcome screen).
   - Navigate to the extracted "SmartStock" folder and select it.
   - Click OK.

3. WAIT FOR GRADLE SYNC
   - Android Studio will automatically start downloading
     Gradle and all project dependencies.
   - This may take 3-10 minutes on first open depending
     on internet speed.
   - Wait until the bottom status bar shows "BUILD SUCCESSFUL"
     or the sync indicator finishes.
   - If prompted to install missing SDK components, click
     "Install" and let it complete.

4. BUILD THE PROJECT
   - Go to Build > Make Project (or press Ctrl+F9).
   - Wait for the build to complete successfully.
   - The first build may take 2-5 minutes due to annotation
     processing (Hilt/KSP).

5. RUN THE APP
   - Connect a physical Android device via USB with
     Developer Mode and USB Debugging enabled.
     OR
   - Create an emulator: Tools > Device Manager > Create Device
     (recommended: Pixel 6, API 35).
   - Click the green Run button (Shift+F10) or go to
     Run > Run 'app'.
   - The app will install and launch on the device/emulator.

TROUBLESHOOTING
------------------------------------------------------------

Problem: Gradle sync fails with "Could not resolve dependencies"
Solution: Ensure you have an active internet connection.
          Go to File > Invalidate Caches > Invalidate and Restart.

Problem: "SDK not found" or "SDK location not set"
Solution: Go to File > Project Structure > SDK Location
          and set the Android SDK path.
          Typically: C:\Users\<username>\AppData\Local\Android\Sdk

Problem: Build fails with KSP or annotation processing errors
Solution: Go to File > Invalidate Caches > Invalidate and Restart.
          Then Build > Clean Project, followed by Build > Rebuild Project.

Problem: "No device connected" when trying to run
Solution: For physical device - enable Developer Options and
          USB Debugging in the phone's Settings > About Phone
          (tap Build Number 7 times).
          For emulator - create one via Tools > Device Manager.

Problem: App crashes on launch
Solution: Check the Logcat panel (View > Tool Windows > Logcat)
          for error details. Ensure the device is API 26 or higher.


PROJECT STRUCTURE
------------------------------------------------------------
SmartStock/
  app/
    src/main/java/com/example/smartstock/
      core/           - Network and sync infrastructure
      data/           - Room database, entities, DAO, repository
      di/             - Hilt dependency injection module
      ui/             - Jetpack Compose screens and ViewModels
      SmartStockApp.kt - Hilt application entry point
    src/main/res/     - Resources (XML layouts, drawables, values)
    build.gradle.kts  - App-level build configuration
  gradle/
    libs.versions.toml - Dependency version catalog
  build.gradle.kts    - Project-level build configuration
  settings.gradle.kts - Project settings
  Documentation/      - Presentation files and guides


TECHNOLOGY STACK
------------------------------------------------------------
- Language:           Kotlin
- UI Framework:       Jetpack Compose + Material Design 3
- Architecture:       MVVM + Repository Pattern
- Dependency Inject:  Hilt (KSP-based)
- Local Database:     Room v7 (6 entities, 35 DAO operations)
- State Management:   StateFlow + Kotlin Coroutines
- Image Loading:      Coil Compose 2.7.0
- Barcode Scanning:   ML Kit / CameraX
- Min SDK:            26 (Android 8.0)
- Target SDK:         35 (Android 15)
