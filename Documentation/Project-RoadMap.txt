Project No. 3
App Name: SmartStock+
Subtitle: Offline-First Inventory & Asset Management System

1. Project Overview
SmartStock+ is an Android-based inventory and asset management application designed to help organizations efficiently track equipment, tools, and supplies using a mobile device. The app prioritizes offline-first functionality, ensuring that inventory data remains accessible even without internet connectivity.

The application follows modern Android development practices, using Jetpack Compose, clean architecture, and Room database for reliable local data persistence, with optional cloud synchronization in later stages.

2. Real-World Scenario
Small offices, laboratories, school departments, and student organizations often manage equipment manually using paper logs or spreadsheets. These systems are error-prone, difficult to maintain, and inaccessible during network outages.

SmartStock+ provides a portable and reliable solution that allows users to:
* Record inventory items on-site
* Track availability and condition of assets
* Access inventory data even when offline
* Synchronize records when internet becomes available

This project reflects real-world IT systems used in asset tracking and resource management.
